# Security Policy

If you discover a security issue in SSHipIt, please report it responsibly.

## How to report

- Do not open a public issue for sensitive vulnerabilities.
- Send details privately to the maintainer with:
  - impact
  - reproduction steps
  - affected version/commit
  - suggested fix (optional)

## What to include

- Exact endpoint/flow affected
- Any required config
- Log snippets/screenshots if relevant

## Response goal

Best effort to acknowledge quickly and ship a fix as soon as practical.
