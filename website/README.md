# SSHipIt Website

This folder contains the static marketing site for SSHipIt.

## Deploy on Vercel

1. Import `radhakishan404/sshipit` in Vercel.
2. Set **Root Directory** to `website`.
3. Build command: _(empty)_
4. Output directory: `.`
5. Deploy.

## Local preview

```bash
cd website
python3 -m http.server 4173
```

Open `http://localhost:4173`.
