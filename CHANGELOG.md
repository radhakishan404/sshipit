# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [1.0.0] - 2026-02-18

### Added

- Initial open-source release of SSHipIt
- Self-hosted CI/CD dashboard for Node.js, Next.js, and React
- SSH server management with encrypted credentials
- Bulk production environment variable management
- Deployment queue, history, cancellation, and live logs
- Release-based deployment model (`releases` + `current` symlink)
- Migration support with smart defaults
- Open-source community files:
  - `CONTRIBUTING.md`
  - `CODE_OF_CONDUCT.md`
  - `SECURITY.md`
  - issue and PR templates
