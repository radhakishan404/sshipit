# Support

Thanks for using SSHipIt.

## Where to ask for help

- Bug reports: open an issue using the bug template
- Feature requests: open an issue using the feature template
- Security concerns: follow `SECURITY.md`

## Before opening an issue

- Confirm your SSH connection works from terminal
- Include deployment logs (copy from UI)
- Mention your framework and restart/migration commands
- Share exact error text and reproduction steps
